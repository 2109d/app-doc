입 / 출력

컨트롤러에 의해 바인딩된 화면에 값을 입력 받거나, 출력 받으려는 경우 data-bind속성값에 수행할 형태를 디렉티브로 명시 하여, 해당 디렉티브를 통해 데이터를(모델: vo객체)에 전달한다.

vo객체의 프로토타입은 디렉티브의 변화를 감지 가능한 값으로 observer메소드를 통해 초기화 하며, 파라메터의 유무에 따라 게터와 세터의 형태로 사용한다.

  • $data:화면에 바인딩된 vo객체에 할당된 값
  • $length:vo객체에 의해 할당된 값의 길이. 

  
  

    이미지 경로: 
    
  




app.view('viewName', function(ctrl) {
  ctrl.vo.imgUrl = ctrl.observer('');
});
TEXT 출력text
문자열 원형 그대로의 값을 출력한다. 


ctrl.vo.outText = ctrl.observer('');
ctrl.vo.outText('<span>html 마크업 미적용</span>');
HTML 출력html
문자열의 값을 html의 마크업을 적용한 상태로 출력한다. 


ctrl.vo.outHtml = ctrl.observer('<span>html 마크업 적용</span>');
VALUE 입/출력value
입력 컨트롤 input, textarea에 입/출력되는 값 또는 select의 선택값 사용한다. 





ctrl.vo.inputValue = ctrl.observer('출력');
ctrl.vo.option = ctrl.observer('1');

// 화면으로 부터 입력된 값
var inputValue = ctrl.vo.inputValue();
alert(inputValue);
체크 상태checked
체크박스, 라디오 버튼의 체크된 값에 사용된다. 선택된 상태가 다수 존재하는 체크박스의 경우 배열을 사용한다. 







ctrl.vo.multi = ctrl.observer(['1', '2']);
ctrl.vo.single = ctrl.observer('1');
속성값attr
엘리먼트의 속성값을 변경할때 사용한다. 화면상에 해당 속성값을 변경시 JSON타입으로 기입한다. 연결되는 변수값을 JSON객체로 사용하여 여러건의 속성을 추가 가능하다. 객체의 프로토타입에 대한 값을 observer를 사용하지 않은 경우 변경되지 않는 값으로 1번만 출력된다. 








ctrl.vo.id = ctrl.observer('id_sample');
// JSON객체 사용
ctrl.vo.imgAttr = {
  title: ctrl.observer('image'),
  href: ctrl.observer('/image.jpg')
};
// 화면의 속성값을 vo.toggle을 통해 변경할 수 없음
ctrl.vo.toggle = {
  title: 'image',
  href: '/image.jpg'
};
클래스css
엘리먼트의 클래스를 추가 또는 삭제 한다. boolean값으로 클래스의 추가 삭제가 가능하며, JSON객체를 사용하여 여러건을 추가 삭제 가능하다. 









ctrl.vo.isOpen = ctrl.observer(true);
// JSON객체 사용
ctrl.vo.toggle = {
  open: ctrl.observer(false),
  close: ctrl.observer(true)
};
스타일style
엘리먼트의 스타일을 변경할때 사용한다. JSON객체를 사용하여 여러건 변경이 가능하다. 









ctrl.vo.display = ctrl.observer('none');
// JSON객체 사용
ctrl.vo.style = {
  display: ctrl.observer('block'),
  color: ctrl.observer('#f00')
};
