입 / 출력

컨트롤러에 의해 바인딩된 화면에 값을 입력 받거나, 출력 받으려는 경우 data-bind속성값에 수행할 형태를 디렉티브로 명시 하여, 해당 디렉티브를 통해 데이터를(모델: vo객체)에 전달한다.

vo객체의 프로토타입은 디렉티브의 변화를 감지 가능한 값으로 observer메소드를 통해 초기화 하며, 파라메터의 유무에 따라 게터와 세터의 형태로 사용한다.

  • $data:화면에 바인딩된 vo객체에 할당된 값
  • $length:vo객체에 의해 할당된 값의 길이. 

      

    

  


  
  

    
TEXT 출력text

    
문자열 원형 그대로의 값을 출력한다.

    

      

      

    

  

  
  
  

    
HTML 출력html

    
문자열의 값을 html의 마크업을 적용한 상태로 출력한다.

    

      

      

    

  


  
  

    
VALUE 입/출력value

    
입력 컨트롤 input, textarea에 입/출력되는 값 또는 select의 선택값에 사용한다.

    

      

      

    

  


  
  

    
체크 상태checked

    
라디오 버튼 또는 체크박스의 체크된 값에 사용된다.
    라디오버튼의 경우 선택된 값이 입/출력된다. 체크박스를 한개만 사용하는 경우에는 boolean값을 사용하며,
    다건의 체크박스인 경우 배열의 값으로 처리 된다.

    

      

      

    

  


  
  

    
속성값attr

    
엘리먼트의 속성값을 변경할때 사용한다. 화면상에 해당 속성값을 변경시 JSON타입으로 기입한다.
    연결되는 변수값을 JSON객체로 사용하여 여러건의 속성을 추가 가능하다.
    객체의 프로토타입에 대한 값을 observer를 사용하지 않은 경우 변경되지 않는 값으로
    화면 랜더링시 초기화를 위해 적용되면 이후 값이 변경되지 않는다.

    

      

      

    

  


  
  

    
클래스css

    
엘리먼트의 클래스를 추가 또는 삭제 한다. boolean값으로 클래스의 추가 삭제가 가능하며,
    JSON객체를 사용하여 여러건을 추가 삭제 가능하다.

    

      

      

    

  


  
  

    
스타일style

    
엘리먼트의 스타일을 변경할때 사용한다. JSON객체를 사용하여 여러건 변경이 가능하다.
