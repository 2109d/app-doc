컨트롤러 기본기능

컨트롤러는 각각 뷰(view), 팝업(popup), 콤포넌트(component) 3종류가 존재 하며, app를 통해 컨트롤러를 등록 후 사용한다. 컨트롤러는 싱글톤(singleton)으로 app의 ctrl속성값에 등록되어 있다. 첫번째 매개변수는 컨트롤러 이름, 두번째 매개변수는 컨트롤러 구현부 프로시저, 세번째는 컨트롤러 메소드 확장시 사용한다.


    

  


  
  

    뷰view(name:string, prodecure:function, extend:function)
    
랜더링이 완료된 영역을 컨트롤 할때 사용한다.
    화면의 data-app-view="ctrl-name"(스네이크 표기법 사용) 속성에 반응하며,
    app의 view('ctrlName', function)(카멜 표기법 사용)으로 등록한다.

    

      

      

    

  


  
  

    팝업popup( name:string, prodecure:function, extend:function )
    
레이어 생성, 추가된 영역을 컨트롤 할때 사용한다. 레이아웃 생성시
    data-app-popup="ctrl-name"(스네이크 표기법 사용) 속성으로 생성되며,
    app의 popup('ctrlName', function)(카멜 표기법 사용)으로 등록한다.
    팝업의 경우 data-app-resource 영역안에 생성되게 된다.

    

      

      

    

  


  
  

    콤포넌트component( name:string, prodecure:function, extend:function )
    
view또는 popup 컨트롤러 안에서 동작하는 영역을 컨트롤 할때 사용한다.
    화면상의 콤포넌트 영역은 주석을 이용하여 콤포넌트의 영역을 지정 하며,
    <!-- ko component:'ctrl-name' -->(스네이크 표기법 사용)
    <!-- /ko --> 사이에 랜더링 된다.
    app의 component('ctrlName', function)(카멜 표기법 사용)으로 등록한다.

    

        

        

    

  


  
시작과 종료

  
  

    시작open(name:string, [param:object]) :promise
    
등록된 컨트롤러를 수행하며, 화면상에 지정된 템플릿으로 랜더링한다.
    첫번째 매개변수는 오픈하려는 컨트롤러명 이며, 두번째 매개변수값에 전달 하고자 하는 파라메터값 이다.
    open메소드는 프로미스를 반환하며, 오픈한 컨트롤러가 종료시 이벤트를 전달 받는다.

  

  
  

    종료close(result:object)
    
오픈된 컨트롤러를 종료할때 사용하며, 매개변수에 결과물을 전달할 수 있다.
    전달하고자 하는 매개변수값은 해당 컨트롤러를 오픈한 주체가되는 컨트롤러의 open함수의 프로미스에 전달된다.
